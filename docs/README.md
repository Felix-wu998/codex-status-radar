# Documentation

This folder contains project material that is useful for GitHub and product development, but is not production app code.

## Sections

- `architecture/` - technical structure, data flow, and repository layout.
- `decisions/` - ADR-style decisions that should remain stable.
- `product/` - PRD, privacy policy drafts, and product scope.
- `research/` - technical spikes, repository reviews, and market notes.

Obsidian remains the working journal. This folder contains the publishable or implementation-relevant subset that should travel with the GitHub repository.
